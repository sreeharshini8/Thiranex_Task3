# 📊 Exploratory Data Analysis on Superstore Dataset

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Superstore dataset to uncover meaningful patterns, trends, correlations, and key business insights using Python.

The analysis includes data cleaning, statistical summaries, visualization techniques, trend analysis, correlation analysis, and identification of important factors affecting sales and profitability.

---

## 🎯 Objective
The main objective of this project is to:

- Analyze retail sales data
- Discover patterns and trends
- Perform statistical analysis
- Identify correlations between variables
- Understand key factors influencing business performance
- Generate meaningful business insights using visualizations

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

---

## 📂 Dataset Information
The dataset contains retail store sales information including:

- Order Details
- Customer Segments
- Product Categories
- Sales
- Profit
- Discounts
- Shipping Information
- Regional Data

## 🔍 Key Insights

- Technology category generated the highest sales and profit.
- Higher discounts negatively impacted profitability.
- Certain regions performed better in terms of sales and profit.
- Consumer segment contributed significantly to overall sales.
- Some states experienced consistent losses despite high sales.

## ✅ Conclusion

This Exploratory Data Analysis helped uncover important sales trends, profitability patterns, and business insights from the Superstore dataset. The project demonstrates how data visualization and statistical analysis can support better business decision-making.
